# coworkers
Coworkers adalah sebuah aplikasi yang dirancang untuk menghubungkan pencari kerja dengan pemberi kerja secara mudah dan efisien. Dengan aplikasi ini, pengguna dapat mencari pekerja berkualitas untuk kebutuhan mereka atau menawarkan diri sebagai pekerja untuk mendapatkan peluang kerja yang sesuai.

## Technology
- Frontend: Flutter (untuk antarmuka pengguna yang responsif dan cross-platform)
- Backend: Flutter (untuk otentikasi, basis data real-time, dan penyimpanan cloud)
- Database: Appwrite
